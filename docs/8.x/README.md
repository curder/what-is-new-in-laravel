<a name="models-directory"></a>
## 模型目录

Laravel 8 的应用程序框架包含一个 `app/Models` 目录。

所有生成器命令都假定模型存在于 `app/Models` 中； 但是，如果此目录不存在，则框架将假定应用程序将模型保存在 `app/` 文件夹中（生成的模型文件路径将保持跟之前一致，在 `app/` 目录下）。
